lfx Learning Engine
H

Harness — Prompt + Playbook Memory

Reflector analyzes failures → Curator resolves conflicts → Playbook accumulates strategies → GEPA evolves prompts via Pareto selection, mutation, crossover

R

Router — Model Selection + Cost Optimization

Tracks per-model quality scores → Routes queries to cheapest model that passes quality threshold → Fallback chains on failure

W

Weights — RL Fine-Tuning

GRPO + custom algorithms via SkyRL/Harbor → gate_for_deploy regression check → Atomic snapshot-rollback on failure

↕ Layer Protocol: forward_backward → optim_step · Content-addressed StateID · Snapshot-Rollback
