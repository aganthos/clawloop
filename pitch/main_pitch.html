Competition

Palantir

End-to-end bespoke intelligence. Huge teams, $1M+ contracts. Not a developer tool — a consulting relationship.

Poetiq ($46M seed)

Black-box meta-system from ex-DeepMind. Recursive self-improvement at test time. No developer control, no weight-level learning. Benchmark-proven, enterprise-unproven.

Adaptive ML

Forward-deployed engineering shop. Embeds ML engineers at $200K+/yr. High-touch, no self-serve product. Cannot scale without headcount.

n8n / Zapier / static platforms

Build agent workflows but never improve them. Ship once, deploy, hope. These are our customers — they need learning but don’t have it.

Aganthos

ClawLoop: open-core developer library. Three learning layers (harness + routing + weights). Regression gate before every rollout. The only closed loop from trace to deploy.
